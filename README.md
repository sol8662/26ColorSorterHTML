## ⚠️ 개발 환경 및 제약 조건 (필수)
- **단일 파일 제약:** 외부 파일(style.css, script.js 등)로 분리하지 마세요. 모든 CSS는 `<style>` 태그 안에, 모든 JavaScript는 `<script>` 태그 안에 포함된 **오직 하나의 단일 HTML 파일(`index.html`)**로만 작성해야 합니다.
- **순수 기술 사용:** React, Vue, Vite, Next.js 등을 절대 사용하지 마세요. 오직 **HTML5, CSS3, 순수 자바스크립트(Vanilla JS)**만 사용해야 합니다.
- **과잉 구현 금지:** 내가 요청한 기능 외에 임의로 새로운 효과나 UI 요소를 추가하지 마세요.

## 요청 사항
중학생인 내가 인공지능 색상 분류 원리를 배울 수 있는 웹사이트를 만들고 싶어. 제목은 'AI 자동 수확 로봇'이야. 화면은 총 4개(시작, 훈련, 게임, 결과)로 구성되어야 해. 

1. **시작 화면:** 카메라 모드와 카드 모드를 선택하고 훈련으로 넘어가는 화면
2. **훈련 화면:** 사과, 바나나, 포도, 오렌지 4가지 과일의 색을 각각 3개씩 수확 로봇에게 학습(저장)시키는 화면
3. **게임 화면:** AI 로봇이 무작위로 지정된 과일을 제한 시간 내에 수확하는 화면
4. **결과 화면:** 수확량 점수와 정확도를 별점으로 보여주는 화면

화면 전환이 기본적으로 작동하는 완성된 단일 HTML 파일 코드를 짜줘. 다른 부가 기능은 넣지 마.
