# HTML 단일 파일 빌드용 단계별 프롬프트 가이드

- HTML 단일파일로 제작

### 📅 1단계 : 웹페이지 기본 뼈대 만들기 (HTML 구조)
- 화면 전환 구조와 레이아웃 뼈대를 잡되, 반드시 단일 HTML 파일로 출력하도록 강제합니다.
- 프롬프트
```
## ⚠️ 개발 환경 및 제약 조건 (필수)
- **단일 파일 제약:** 외부 파일(style.css, script.js 등)로 분리하지 마세요. 모든 CSS는 `<style>` 태그 안에, 모든 JavaScript는 `<script>` 태그 안에 포함된 **오직 하나의 단일 HTML 파일(`index.html`)**로만 작성해야 합니다.
- **순수 기술 사용:** React, Vue, Vite, Next.js 등을 절대 사용하지 마세요. 오직 **HTML5, CSS3, 순수 자바스크립트(Vanilla JS)**만 사용해야 합니다.
- **과잉 구현 금지:** 내가 요청한 기능 외에 임의로 새로운 효과나 UI 요소를 추가하지 마세요.

## 요청 사항
중학생인 내가 인공지능 색상 분류 원리를 배울 수 있는 웹사이트를 만들고 싶어. 제목은 'AI 자동 수확 로봇'이야. 화면은 총 4개(시작, 훈련, 게임, 결과)로 구성되어야 해. 

1. **시작 화면:** 카메라 모드와 카드 모드를 선택하고 훈련으로 넘어가는 화면
2. **훈련 화면:** 사과, 바나나, 포도, 오렌지 4가지 과일의 색을 각각 3개씩 수확 로봇에게 학습(저장)시키는 화면
3. **게임 화면:** AI 로봇이 무작위로 지정된 과일을 제한 시간 내에 수확하는 화면
4. **결과 화면:** 수확량 점수와 정확도를 별점으로 보여주는 화면

화면 전환이 기본적으로 작동하는 완성된 단일 HTML 파일 코드를 작성해줘. 다른 부가 기능은 넣지 마.
```
### 🎨 2단계: 미래지향적인 SF 스타일 입히기 (CSS 디자인)
- 기존 코드의 구조를 유지하면서 <style> 태그 내부에 스타일만 업데이트합니다.
- 프롬프트
```
## ⚠️ 제약 조건
- 코드를 새로 적성하지 말고, 기존 HTML 구조를 유지한 채 `<style>` 태그 내부와 클래스 디자인만 업데이트해 줘.
- 여전히 외부 파일 분리 없이 **하나의 HTML 파일**로 출력해야 해.

## 요청 사항
앞서 만든 HTML 뼈대에 CSS 스타일을 적용해 줘. 컨셉은 '스마트팜의 첨단 로봇 시각 센서' 느낌이야. 

- 배경은 어두운 청록색 계열로 하고, 테두리나 강조선은 네온 청록색(#6FFFB0)을 사용해 줘.
- 전체적으로 깔끔한 카드 스타일 디자인에 테두리를 둥글게 만들어 줘.
- 훈련 화면과 게임 화면에는 로봇이 색상을 스캔하는 듯한 느낌을 주는 위아래로 움직이는 '스캔 라인 애니메이션'을 추가해 줘.
```
### 🧠 3단계: 로봇의 눈과 두뇌 만들기 (카메라 연동 & AI 알고리즘)
- 웹캠을 연동하고, 자바스크립트 배열과 함수를 통해 분류 알고리즘을 이식합니다.
- 프롬프트
```
## ⚠️ 제약 조건
- 외부 AI 라이브러리(TensorFlow.js 등)를 사용하지 말고, 오직 **순수 자바스크립트(Vanilla JS)의 수학 연산**만으로 알고리즘을 구현해 줘.
- 코드는 계속해서 하나의 HTML 파일로 합쳐져 있어야 해.

## 요청 사항
이제 핵심인 AI 기능을 넣을 차례야. HTML5 웹캠(`navigator.mediaDevices.getUserMedia`)을 연동해서 화면 중앙의 특정 영역(타겟 박스)에 비치는 색상의 평균 RGB 값을 추출하는 코드를 `<script>` 안에 짜줘.

그리고 인공지능 역할을 할 분류 알고리즘을 만들어 줘. 과일별로 저장된 3개 색상 데이터의 평균값(중심점)을 구한 뒤, 새로 입력된 색상과의 거리가 가장 가까운 과일로 판정하는 '유클리드 거리 기반 알고리즘'을 순수 자바스크립트 함수로 구현해서 연동해 줘.
```

### 🎮 4단계: 게임 요소와 규칙 추가하기 (완성 및 디버깅)
- 타이머, 점수, 콤보 등의 시스템을 붙여 최종 단일 웹 애플리케이션을 완성합니다.
- 프롬프트
```
## ⚠️ 제약 조건
- 최종 단계야. 다른 개발 환경(Vite/React 등)으로 절대 전환하지 말고, 브라우저에서 바로 실행 가능한 **최종 완성형 단일 HTML 파일 코드 전체**를 한 번에 출력해 줘.

## 요청 사항
마지막으로 이 프로그램을 재미있는 게임으로 완성해 줘! 카드 모드와 카메라 모드의 다른 형태로 제작해야해!

- **라운드:** 총 12라운드로 진행되고, 라운드마다 AI 로봇이 찾아야 할 목표 과일이 무작위로 지정돼야 해.
- **타이머:** 라운드마다 제한 시간이 시각적으로 줄어드는 타이머 바를 만들어 줘. 시간이 다 되면 다음 라운드로 넘어가야 해.
- **콤보 및 점수:** 연속으로 맞추면 콤보(`x1`, `x2`)가 쌓여 점수가 더 높게 올라가고, 틀리면 콤보가 리셋되게 해줘.
- **카드 모드**: 사용자가 직접 과일 이미지를 가져오거나 클립보드에 있는 이미지를 붙여넣기 하여 점수를 맞추는 형태로 제작해야 해.
- **카메라 모드**: 사용자가 직접 카메라에 과일을 촬영하면 맞추는 형태로 제작해야 해.
- 게임이 끝나면 정확도를 백분율(%)로 계산하고, 결과에 따라 별점(`★`)을 다르게 부여하는 로직을 추가해서 최종 결과 화면을 완성해 줘.
```
